# CLI-утилита для отслеживания бюджета

Это простая CLI-утилита для управления личным бюджетом.

## Установка

```bash
npm install
```

## Доступные команды

- `npm run build`: Компиляция TypeScript в JavaScript.
- `npm run start`: Запуск приложения.
- `npm run dev`: Запуск компилятора TypeScript в режиме отслеживания изменений.



## Использование



#### Добаление счета

- в гланом меню выбирите `Добавить счет`
- введите имя счета и нажмите Enter



#### Работа со счетом

- в главном меню выберите `Выбрать счет`
- использую клавиши управления курсором выберите счет и нажмите Enter



#### Добавление транзакции

- в меню счета выберите `Добавить транзакцию`

- выберите тип транзакции: доход (income) или траты (expense)

- введите сумму (должна быть положительным числом)

- введите описание

  

#### Удаление транзакции

- в меню счета выберите `Удалить транзакцию`

- выберите выберите транзакцию для удаления и нажмите Enter

  

#### Экспорт транзакций

- в меню счета выберите `Экспорт в CSV`

- ввелите имя файла для экспорта и нажмите Enter
